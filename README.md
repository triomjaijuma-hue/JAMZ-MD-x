JAMZ-MD v6.0.0 WhatsApp bot designed for Railway FREE tier.

## Features
- Baileys Multi-Device
- Pair Code Login
- Dynamic Plugin System
- 269+ Commands
- AI Powered (OpenAI)

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env` file with:
   - `OPENAI_API_KEY`: Your OpenAI API key
   - `BOT_OWNER_WA_ID`: Your WhatsApp ID (e.g., 256706106326)
4. Start the bot: `npm start`
5. Enter your phone number when prompted to get the pair code.
